Step 2: API Endpoints
Marketstack offers several endpoints:

End-of-Day Data: Retrieve daily stock market data.
Intraday Data: Retrieve intraday and real-time market data.
Tickers: Get information about stock ticker symbols.
Exchanges: Retrieve information about supported exchanges.
Currencies: Retrieve information about supported currencies.
Timezones: Retrieve information about supported timezones.
Base URL: All API requests start with:

http://api.marketstack.com/v1/
You can also get started quickly with Postman by forking the official Marketstack API Postman Collection.


Step 3: Make API Requests
Try simple requests for end-of-day, intraday, and static data endpoints. Append your access_key to any endpoint.

End-of-Day Data Endpoint

http://api.marketstack.com/v1/eod?access_key=YOUR_ACCESS_KEY&symbols=AAPL
// Optional parameters:
&sort=DESC
&date_from=YYYY-MM-DD
&date_to=YYYY-MM-DD
&limit=100
&offset=0
Intraday Data Endpoint

http://api.marketstack.com/v1/intraday?access_key=YOUR_ACCESS_KEY&symbols=AAPL
// Optional parameters:
&interval=1h
&sort=DESC
&date_from=YYYY-MM-DD
&date_to=YYYY-MM-DD
&limit=100
&offset=0

http://api.marketstack.com/v1/tickers?access_key=YOUR_ACCESS_KEY

More parameters are available for each endpoint. Refer to the API Documentation for full details.

Step 4: First API Call
We recommend starting with the End-of-Day Data endpoint, as it is the primary method for retrieving daily stock data. Use the symbols parameter to query one or multiple ticker symbols (comma-separated).
